Add a new app and allow it only on weekends

A new game just got installed. Here's how to keep it off during the week and free it up Saturday and Sunday — without changing anything else you've already set up.

← All guides

This uses the same two-profile idea as a school-hours setup, just inverted: your everyday (default) profile blocks the new app, and a "Weekend" profile — active only Saturday and Sunday — doesn't.

If you've only got one profile so far

1

Block the new app on your existing profile

Go to Setup → Manage profiles → tap your profile → find the new app under "Apps to block" and turn it on. Save. It's now blocked every day, all the time.

2

Create a second profile called Weekend

Setup → Manage profiles → + Create a new profile. Pick which apps this one blocks — leave the new app unticked here, since this profile is what "free time" looks like. You can block other things on it too if you want (e.g. still no social media), just not the app you're freeing up.

3

Set its schedule

Leave "Set profile to default" off. Toggle on Saturday and Sunday only, turn on "All day." Save.

4

Done

If the phone's already bricked, that's it — Weekend switches in automatically every Saturday morning and switches back to your everyday profile Monday morning. If it isn't bricked yet, tap BRICK on Home once to arm it.

If you already have a scheduled profile (e.g. School Day)

You don't need a third profile. Just add the new app to whichever profile represents "the rest of the time" — your default profile — and leave it off your School Day profile's block list if you're happy for it to be available outside school hours too. If you specifically want it weekends-only rather than "anytime outside school," follow the steps above to add a dedicated Weekend profile alongside School Day — Brickify supports more than one scheduled profile at once.

Why this works: a profile's schedule only decides when it's active — the app list is entirely separate. The new app being blocked or not is purely down to whether it's ticked on the profile that's active at that moment. Nothing else you've configured changes.

On Android, you'll get an email when a new app is installed, if you've added your address in Setup. iOS can't detect new installs at all — that's exactly why blocking the App Store is worth doing on any profile you want to hold firm.
