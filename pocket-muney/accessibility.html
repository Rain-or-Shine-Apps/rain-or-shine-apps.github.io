We want Pocket Muney to work well for everyone. Here is what the current version supports and what we are still working on.

👁 Screen Reader (VoiceOver & TalkBack)

Key interactive elements in Pocket Muney — including child tabs, chore buttons, the Pay Day button, payout history, and the manual carrot input — have descriptive accessibility labels. A screen reader will announce the chore name, reward value, and whether it has been completed today.

iPhone / iPad — VoiceOver Settings → Accessibility → VoiceOver
Android — TalkBack Settings → Accessibility → TalkBack

🔤 Text Size

Text in Pocket Muney will grow and shrink with your device's font size setting. At very large sizes some layouts may not adapt perfectly, but all text will remain readable.

iPhone / iPad Settings → Display & Brightness → Text Size
Android Settings → Display → Font Size

⚡ Reduce Motion

Pocket Muney checks your device's Reduce Motion setting. When it is turned on, the Pay Day coin animation is skipped.

iPhone / iPad Settings → Accessibility → Motion → Reduce Motion
Android Settings → Accessibility → Remove Animations

🌗 Light & Dark Mode

Pocket Muney currently uses a fixed colour scheme and does not follow your device's light or dark mode setting. We plan to add dark mode support in a future update.
